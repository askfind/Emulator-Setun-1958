# 1. Описание бинарного формата виртуальной перфоленты для загрузки и запуска троичных программ для "СЕТУНЬ"

Формат файла виртуальной перфоленты с расширением 'FileName.tsb'

struct PaperTSBDesc {
    char tag[4] // "tsb " - тэг описание типа файла 4 байта
    uint16_t len; // размер массива (файла) с тритами
    uint8_t * trits; // 3 трита и контроль четности
}

—----------------------------------------
Таб 1. Кодирование комбинация бит и тритов
-----------------
Биты Трит
b1b0 t0
—---------------
10 +
00 0
01 -
—--------------
Описание поля бит элемента тип данных uint8_t elm
бинарного файла виртуальной перфоленты.

—---—-----------------------------------------------
Таб 2. Элемент бинарного файла виртуальной перфоленты
uint8_t data (поле из бит для кодирования 3-х трит)
[t2t1t0] закодированный триты из 6 бит [b5b4b3b2b1b0]
—----------------------------------------------------
data[b7...b0]
—----------------------------------------------------
b7 b6 b5b4 b3b2 b1b0
0 parity trit2 trit1 trit0
—----------------------------------------------------
Примечание:
- Кодирование трита бинарных кодом '11' является недопустимым и используется для контроля валидности трита.
- При загрузке выполнить аврийный останов загрузки бинарного варианта
виртуальной перфоленты.
